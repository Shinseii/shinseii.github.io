Balise Div ¶

Tout d'abord, il faut créer un conteneur qui contiendra les élements.
On crée une balise div avec id="conteneur", dans cette balise, nous créons d'autres balises div avec class="element" qui ont tous la même classe, mais il est possible d'en choisir des différents. Ces div sont autour des sections ou bloc de textes que vous voulez organiser. Vous pourrez ainsi les déplacer en colonnes, en lignes, etc...facilement.

  • < div >: Balise universelle.

    • <div id="conteneur">
	<div class="element">Elément 1</div>
	<div class="element">Elément 2</div>
	<div class="element">Elément 3</div>
</div>

Conteneur ¶

Dans le fichier .css, on applique à #conteneur différents paramètres influant sur tous les div à l'intérieur du DIV avec id="conteneur". On manie tous les blocs de div, toutes les sections signalées par div en modifiant #conteneur.

  • Bloc côte à côte 
    • #conteneur
{
	display:flex;
}
  • Direction d'organisation des blocs : row (ligne), column (colonne), row-reverse (ligne ordre inversé), column-reverse (colonne ordre inversé)
    • #conteneur
{
	display: flex;
	flex-direction: column;
}
  • Retour à la ligne : nowrap (les elements se resserrent au max), wrap (element passe à la ligne quand plus de place), wrap-reverse
    • #conteneur
{
	display: flex;
	flex-wrap: wrap;
}
  • Alignemer sur l'axe principal : flex-start (alignés au début), flex-end (alignés à la fin), center (alignés au centre), space-between (espace entre chaque bloc, sur tout l'espace), space-around (même chose + espace extrémités)
    • #conteneur
{
	display: flex;
	justify-content: space-around;
}
  • Aligner sur l'axe secondaire : stretch (défaut), flex-start (alignés au début), flex-end (alignés à la fin), center (alignés au centre), baseline (alignés sur la ligne de base)
    • #conteneur
{
	display: flex;
	justify-content: center;
	align-items: center;
}

Elements ¶

On gère désormais un seul ou un nombre fixe d'éléments. Dans notre cas, ils portent tous la même classe.

  • Aligner un seul élement (ici l'élement 2 à la fin)
    • #conteneur
{
	display: flex;
	flex-direction: row;
	justify-content: center;
	align-items: center;
}

.element:nth-child(2)
{
	align-self: flex-end;
	background-color: blue;
}
  • Gérer l'ordre des élements
    • .element:nth-child(1)
{
	order: 3;
}
.element:nth-child(2)
{
	order: 1;
}
.element:nth-child(3)
{
	order: 2;
}
  • Allonger un bloc, le réduire (ici, l'élement 1 est 2 fois plus long)
    • .element:nth-child(1)
{
	flex: 2;
}
.element:nth-child(2)
{
	flex: 1;
}
.element:nth-child(3)
{
	flex: 1;
}
