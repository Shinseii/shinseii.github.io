Propriétés texte ¶

  • font : Super propriété de police, combine : font-weight, font-style, font-size, font-variant, font-family.

    • font: bold italic 1.3em normal Times New Roman;

  • font-family : Nom de police.

    • font-family: police1, police2, police3, serif, sans-serif, monospace...;

  • @font-face : Police personnalisée (création).

    • @font-face: Nom et source de la police...;

  • font-size : Taille du texte.

    • font-size: 1.3em, 16px, 120%...;

  • font-weight : Gras.

    • font-weight: bold, normal;

  • font-style : Italique.

    • font-style: italic, oblique, normal;

  • font-variant : Petites capitales. 

    • font-variant: small-caps, normal;

  • text-decoration : Soulignement, ligne au dessu, barré, clignotant.

    • text-decoration: underline, overline, line-through, blink, none;

  • text-transform : Capitales.

    • text-transform: capitalize, lowercase, uppercase;

  • text-align : Alignement horizontal.

    • text-align: left, center, right, justify;

  • text-indent : Alinéa.

    • text-indent: 25px...;

  • text-shadow : Ombre de texte.

    • text-shadow: 5px 5px 5px blue; (horizontale, verticale, fondu, couleur)

  • vertical-align : Alignement vertical (cellules tableau ou éléments inline-block uniquement).

    • vertical-align: baseline, middle, sub, super, top, bottom;

  • white-space : Césure.

    • white-space: pre, nowrap, normal;

  • word-wrap : Césure forcée.

    • word-wrap: break-word, normal;

Propriétés couleur et fond ¶

  • background : Super propriété du fond, combine : background-image, background-repeat, background-attachment, background-position.

    • background: url('image.png) repeat fixed center;

  • background-color : Couleur de FOND.

    • background-color: couleur, rgb(x,y,z, transparence), rgba(x,y,z, transparence), #CF1A20...;

  • background-image : Image de fond.

    • background-image: url('image.png');

  • background-attachment : Fond fixe ou non.

    • background-attachment: fixed, scroll;

  • background-repeat : Répétition du fond.

    • background-repeat: repeat-x, repeat-y, no-repeat, repeat;

  • background-position : Position du fond.

    • background-position: (x,y), top, center, bottom, left, right;

  • color : Couleur du texte.

    • color: nomcouleur, rgb(x,y,z, transparence), rgba(x,y,z, transparence), #CF1A20...;

  • opacity : Transparence.

    • opacity: 0.5...;

Propriétés des boites ¶

  • margin : Super propriété de marge, combine : margin-top, margin-right, margin-bottom, margin-left.

    • margin: 23px 5px 23px 5px; (haut, droite, bas, gauche)

  • margin-top (-left,-right,-bottom) : Marge en haut, à gauche, à droite, en bas.

    • margin-top (-left,-right,-bottom): 23px;

  • padding : Super propriété de marge intérieure, combine : padding-top, padding-right, padding-bottom, padding-left.

    • padding: 23px 5px 23px 5px; (haut, droite, bas, gauche)

  • padding-top (-left,-right,-bottom) : Marge intérieure en haut, à gauche, à droite, en bas.

    • padding-top (-left,-right,-bottom): 23px;

  • border : Super-propriété de bordure, combine : border-width, border-color, border-style. Existe aussi en version : border-top, border-right, border-bottom, border-left..

    • border: 3px solid black;

  • border-width : Epaisseur de bordure.

    • border-width: 3px;

  • border-color : Couleur de bordure.

    • border-color: nom, rgb(rouge,vert,bleu), rgba(rouge,vert,bleu,transparence), #CF1A20...;

  • border-radius : Bordure arrondie.

    • border-radius: 10px;

  • border-style : Type de bordure.

    • border-style: solid, dotted, dashed, double, groove, ridge, inset, outset;

  • width : Largeur.

    • width: 150px, 80%, 1em;

  • height : Hauteur.

    • height: 150px, 80%, 1em;

  • min-width (-height) : Largeur ou Hauteur minimale.

    • min-width (-height): 150px, 80%, 1em;

  • max-width (-height) : Largeur ou Hauteur maximale.

    • max-width (-height): 150px, 80%, 1em;

  • box-shadow : Ombre de boîte.

    • box-shadow: 6px 6px 0px black; (horizontale, verticale, fondu, couleur)

Propriétés position et affichage ¶

  • display : Type d'élément (block, inline, inline-block, none...).

    • display: block, inline, inline-block, table, table-cell, flex...;

  • visibility : Visibilité.

    • visibility: visible, hidden;

  • clip : Affichage d'une partie de l'élément.

    • clip: rect(0px, 60px, 30px, 0px); (haut, droite, bas, gauche)

  • overflow : Comportement en cas de dépassement.

    • overflow: auto, scroll, visible, hidden;

  • float : Flottant.

    • float: left, right, both, none;

  • clear : Arrêt d'un flottant.

    • clear: left, right, both, none;

  • position : Position.

    • position: relative, absolute, static;

  • top : Position par rapport au haut.

    • top: 20px;

  • bottom : Position par rapport au bas.

    • bottom: 20px;

  • left : Position par rapport à la gauche.

    • left: 20px;

  • right : Position par rapport à la droite.

    • right: 20px;

  • z-index : Ordre d'affichage en cas de superposition. La plus grande valeur est affichée par-dessus les autres.

    • z-index: 10;

Propriétés des listes ¶

  • list-style : Super-propriété de liste, combine : list-style-type, list-style-position, list-style-image.

    • list-style: disc inside url('puce.png');

  • list-style-type : Type de liste.

    • list-style-type: disc, circle, square, decimal, lower-roman, upper-roman, lower-alpha, upper-alpha, none;

  • list-style-position : Position en retrait.

    • list-style-position: inside, outside;

  • list-style-image : Puce personnalisée.

    • list-style-image: url('puce.png')

Propriétés des tableaux ¶

  • border-collapse : Fusion des bordures.

    • border-collapse: collapse, separate;

  • empty-cells : Affichage des cellules vides.

    • empty-cells: hide, show;

  • caption-side : Position du titre du tableau.

    • caption-side: bottom, top;

Autres propriétés ¶

  • cursor : Curseur de souris.

    • cursor: crosshair, default, help, move, pointer, progress, text, wait, e-resize, ne-resize, auto...;

Hover, active, focus, visited ¶

  • hover : Apparence au survol des liens.

    • a:hover
    {
       text-decoration: underline;
       color: green;
    }

  • active : Au moment du clic.

    • a:active
    {
       background-color: #FFCC66;
    }

  • focus : Lorsque l'élément est sélectionné.

    • a:focus
    {
       background-color: #FFCC66;
    }

  • visited : Lorsque le lien a déjà été visité.

    • a:visited
    {
       color: #FFCC66;
    }
